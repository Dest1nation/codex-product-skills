# Exploration Tracker

Track exploration so no context is lost during facilitation and later summarization.

## Track

- Conversation turns: important user reflections and model prompts.
- Facts: confirmed by user or source material.
- Assumptions: inferred to move forward.
- Options: possible product directions.
- Rejected options: considered but not selected.
- Open questions: unresolved and impact-rated.
- Decision history: why a path was chosen.

Use `memory/` templates when the conversation becomes long or the user asks for a formal record.
