# Hypothesis Builder

Use this when the user has a problem statement but the solution is not fixed.

## Hypothesis Template

```markdown
| 假设 | 依据 | 可能方案 | 验证方式 | 风险 |
| --- | --- | --- | --- | --- |
```

## Rules

- Separate user pain, business goal, and solution idea.
- Keep multiple viable options visible until a decision is made.
- Mark confidence as high, medium, or low.
- If a hypothesis affects scope or state design, flag it as blocking before structuring.
